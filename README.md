
Space Tourism Website
Welcome to our Space Tourism website repository! This project aims to provide an immersive and informative experience for users interested in space travel.

Features
Homepage: Engaging landing page with stunning visuals and key information about space travel packages.
Booking System: Secure booking system for users to select their preferred space tour package and make reservations.
Interactive Gallery: Explore breathtaking images and videos of space destinations.
Informational Pages: Learn about different celestial bodies, space exploration history, and future missions.
Contact Page: Contact form for inquiries, feedback, and support requests.
Technologies Used
HTML: Markup language for structuring web pages.
CSS & SCSS: Stylesheets for designing and styling the website.
JavaScript: Used for client-side interactions and animations.
PHP: Backend scripting language for server-side functionalities.

Directory Structure
css/: Contains CSS and SCSS files for styling.
js/: JavaScript files for interactive elements and animations.
img/: Images and videos used in the website.
includes/: PHP files for reusable components like header, footer, and navigation.
pages/: HTML files for different pages of the website.
assets/: Additional assets such as fonts and icons.
